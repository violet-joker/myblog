# ffmpeg操作音视频文件

## 转换音频

将下载的mp3文件转换成wav文件，并设置单声道

```shell
ffmpeg -i input.mp3 -ac 1 output.wav

-sample_fmt u8 
-c:a pcm_u8
```

## 转换视频

将mp4文件转换成便于esp32解析的avi文件

```c++
ffmpeg -i input.mp4 -s 320x240 -c:v mjpeg -q:v 10 -ac 1 -f avi output.avi

# 解析成RGB565裸二进制文件

ffmpeg -i input.avi -an -r 24 -pix_fmt rgb565le -f rawvideo -vsync cfr output.bin

-an  # 剔除音频
-r   # 设置帧率
-vsync cfr  # 强制恒定帧率，避免帧丢失、帧重复
```

## 合并video.m4s和audio.m4s

```shell
ffmpeg -i video.mp4 -i audio.mp4 -codec copy output.mp4
```

## 视频切片

```shell
ffmpeg -i input.mp4 -ss 00:00:00 -to 03:48:55 -c copy output.mp4
```
